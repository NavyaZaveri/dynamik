# dynamik

Dynamik is a dynamically typed language, written in Kotlin.   


### Features

- [X] Types - Strings/Doubles

- [x] Arithmetic Expressions / String manipulation 

- [x] (Im)mutable bindings 

- [x] If/Else

- [x] For/While loops

- [x] Functions 

- [x] Built-in memoization 

- [x] Type checks

- [ ] Par statements 

- [ ] Collections (list, map)

- [x] Repl
