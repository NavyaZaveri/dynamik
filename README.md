# dynamik

Inspired by ths simplicity of Lox, Dynamik is a similar dynamically typed  language with additional features tacked on.  


### Features

- [x] Arithmetic Expressions 

- [x] (Im)mutable bindings 

- [x] If/Else

- [x] For/While loops

- [x] Functions 

- [x] Built-in memoization 

- [x] Type checks

- [ ] Par statements 

- [ ] Collections

- [x] Repl
